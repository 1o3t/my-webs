Python数据结构

掌握Python的内置数据结构，包括列表、元组、字典和集合，这些是Python编程中最常用的数据组织方式。

列表 (List)

列表是Python中最常用的数据结构之一，它是一个有序的、可变的元素集合，可以包含不同类型的元素。 

# 创建列表
fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4, 5]
mixed = ["apple", 1, True, 3.14]

# 访问列表元素
print(f"第一个元素: {fruits[0]}")
print(f"最后一个元素: {fruits[-1]}")

# 切片操作
print(f"前两个元素: {fruits[:2]}")
print(f"从第二个元素开始: {fruits[1:]}")

# 修改列表
fruits[1] = "grape"
print(f"修改后的列表: {fruits}")

# 添加元素
fruits.append("orange")  # 在末尾添加
fruits.insert(1, "pear")  # 在指定位置插入
print(f"添加元素后: {fruits}")

# 删除元素
fruits.remove("apple")  # 删除指定元素
popped = fruits.pop()  # 删除并返回最后一个元素
print(f"删除元素后: {fruits}")
print(f"弹出的元素: {popped}")

# 列表方法
print(f"列表长度: {len(fruits)}")
print(f"元素是否在列表中: {'banana' in fruits}")
print(f"元素出现次数: {fruits.count('cherry')}")
fruits.sort()  # 排序
print(f"排序后: {fruits}")
fruits.reverse()  # 反转
print(f"反转后: {fruits}")

元组 (Tuple)

元组是一个有序的、不可变的元素集合，一旦创建就不能修改。元组使用圆括号表示。 

# 创建元组
tuple1 = (1, 2, 3)
tuple2 = "a", "b", "c"  # 可以不用括号
single_tuple = ("apple",)  # 单个元素的元组需要加逗号

# 访问元组元素
print(f"第一个元素: {tuple1[0]}")
print(f"最后一个元素: {tuple1[-1]}")

# 切片操作
print(f"前两个元素: {tuple1[:2]}")

# 元组是不可变的，以下操作会报错
# tuple1[0] = 4  # 会抛出TypeError

# 元组方法
print(f"元组长度: {len(tuple1)}")
print(f"元素是否在元组中: {2 in tuple1}")
print(f"元素出现次数: {tuple1.count(2)}")
print(f"元素索引: {tuple1.index(2)}")

# 元组的优点
# 1. 占用内存小
# 2. 访问速度快
# 3. 可以作为字典的键
# 4. 可以在集合中使用

字典 (Dictionary)

字典是一个无序的键值对集合，使用键来访问值。字典使用花括号表示，键和值之间用冒号分隔。 

# 创建字典
student = {"name": "Alice", "age": 20, "major": "Computer Science"}
empty_dict = {}

# 访问字典值
print(f"学生姓名: {student['name']}")
print(f"学生年龄: {student.get('age')}")
print(f"不存在的键: {student.get('grade', 'A')}")  # 提供默认值

# 修改字典
student["age"] = 21  # 修改现有键的值
student["grade"] = "A"  # 添加新的键值对
print(f"修改后的字典: {student}")

# 删除键值对
del student["major"]  # 删除指定键
popped = student.pop("age")  # 删除并返回指定键的值
print(f"删除后的字典: {student}")
print(f"弹出的值: {popped}")

# 字典方法
print(f"字典键: {list(student.keys())}")
print(f"字典值: {list(student.values())}")
print(f"字典项: {list(student.items())}")
print(f"键是否存在: {'name' in student}")

# 遍历字典
print("\n遍历字典:")
for key, value in student.items():
    print(f"{key}: {value}")

集合 (Set)

集合是一个无序的、不重复的元素集合，使用花括号表示。集合支持数学中的集合运算，如并集、交集、差集等。 

# 创建集合
set1 = {1, 2, 3, 4, 5}
set2 = set([3, 4, 5, 6, 7])  # 从列表创建

# 添加元素
set1.add(6)
set1.update([7, 8, 9])  # 添加多个元素
print(f"添加元素后: {set1}")

# 删除元素
set1.remove(9)  # 删除指定元素，如果不存在会报错
set1.discard(8)  # 删除指定元素，如果不存在不会报错
popped = set1.pop()  # 随机删除并返回一个元素
print(f"删除元素后: {set1}")
print(f"弹出的元素: {popped}")

# 集合运算
print(f"集合1: {set1}")
print(f"集合2: {set2}")
print(f"并集: {set1 | set2}")
print(f"交集: {set1 & set2}")
print(f"差集: {set1 - set2}")
print(f"对称差集: {set1 ^ set2}")

# 集合方法
print(f"集合长度: {len(set1)}")
print(f"元素是否在集合中: {3 in set1}")

# 集合推导式
numbers = {x for x in range(10) if x % 2 == 0}
print(f"偶数集合: {numbers}")

学习提示

  • 列表是可变的，适合需要频繁修改的数据
  • 元组是不可变的，适合存储不需要修改的数据
  • 字典使用键值对存储数据，适合通过键快速查找值
  • 集合自动去重，适合需要唯一元素的场景
