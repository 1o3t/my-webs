Python基础语法

学习Python的基本语法规则，包括变量、数据类型、运算符和控制流程，为后续的编程学习打下坚实基础。

变量和数据类型

Python中的变量不需要声明类型，直接赋值即可。Python支持多种数据类型，包括整数、浮点数、字符串、布尔值等。 

# 变量赋值
name = "Python"
age = 32
pi = 3.14159
is_awesome = True

# 打印变量
print(f"名称: {name}")
print(f"年龄: {age}")
print(f"π: {pi}")
print(f"是否 awesome: {is_awesome}")

# 数据类型
print(f"name的类型: {type(name)}")  # str
print(f"age的类型: {type(age)}")    # int
print(f"pi的类型: {type(pi)}")     # float
print(f"is_awesome的类型: {type(is_awesome)}")  # bool

运算符

Python支持多种运算符，包括算术运算符、比较运算符、逻辑运算符等。 

# 算术运算符
a = 10
b = 3
print(f"a + b = {a + b}")  # 加法
print(f"a - b = {a - b}")  # 减法
print(f"a * b = {a * b}")  # 乘法
print(f"a / b = {a / b}")  # 除法
print(f"a // b = {a // b}")  # 整数除法
print(f"a % b = {a % b}")  # 取模
print(f"a ** b = {a ** b}")  # 幂运算

# 比较运算符
print(f"a > b: {a > b}")
print(f"a < b: {a < b}")
print(f"a == b: {a == b}")
print(f"a != b: {a != b}")

# 逻辑运算符
x = True
y = False
print(f"x and y: {x and y}")
print(f"x or y: {x or y}")
print(f"not x: {not x}")

条件语句

条件语句用于根据条件执行不同的代码块，Python支持if、elif和else语句。 

# 简单条件语句
score = 85
if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")
elif score >= 60:
    print("及格")
else:
    print("不及格")

# 嵌套条件语句
age = 18
if age >= 18:
    print("成年人")
    if age >= 60:
        print("老年人")
else:
    print("未成年人")

循环语句

循环语句用于重复执行代码块，Python支持for循环和while循环。 

# for循环
print("for循环示例:")
for i in range(5):
    print(f"数字: {i}")

# 遍历列表
fruits = ["apple", "banana", "cherry"]
print("\n遍历列表:")
for fruit in fruits:
    print(f"水果: {fruit}")

# while循环
print("\nwhile循环示例:")
counter = 0
while counter < 5:
    print(f"计数器: {counter}")
    counter += 1

# break和continue
print("\n使用break:")
for i in range(10):
    if i == 5:
        break
    print(f"数字: {i}")

print("\n使用continue:")
for i in range(10):
    if i % 2 == 0:
        continue
    print(f"奇数: {i}")
